# Form Input
A JavaScript Project from Brad Traversy's 50 Projects in 50 Days. This project #8 is a form input. On focus, the labels translate on the Y axis over a duration of 0.3s.

![Form Input](screenshot.jpg)

## Built With
- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Install
```
    $ git clone git@github.com:samiiexx/sam-js-8.git
    $ cd main
```
